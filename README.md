PicoMate is an simple Launcher for bas-, bmp-, wav- and txt-files.

KEY OVERVIEW

up/down   Move cursor"
ENTER     Open file/folder"
D         Switch drive"
F1        Show this help"
N         Create folder"
DEL       Delete file/folder"
C         Copy file"
X         Cut file"
V         Paste into folder"
R         Rename Folder"
F         Filter all/.bas"
